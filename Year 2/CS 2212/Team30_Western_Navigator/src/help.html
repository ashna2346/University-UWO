1. Introduction

Western Navigator was built to help students, staff, and visitors conveniently navigate through buildings on campus. The app hosts a suite of functions to aid with the same.


2. Tool Menus

a. Buildings

This menu allows you to cycle between available campus buildings.

Selecting a building from this list will load the respective maps and it's POIs.


b. Favourites

The app provides users with an option to star their frequently visited POI. Starred POI can be accessed from this dropdown menu.

Clicking on a starred POI from the dropdown will highlight it and center it on the screen.


c. POIs

The POI menu presents users with 2 options-

  1. Discover:

    Opens the discover panel on the side, which displays a list of all the POI available in the current map. POIs are further classified into different categories. Personal POI are also displayed in the discover panel. Discover panel stays open until manually closed by clicking on the 'X' icon.

    Clicking on a POI in this panel will highlight it and center it on the screen.


  2. Create:

    This feature allows users to create their own POI.

    After selecting this option from the menu, users can click anywhere on the map to select the coordinates for their POI. A window opens up to allow user input for the Room Number, Room Name, and Description. These POIs are classified as Personal POI in the discover pane.

d. Layers

This menu displays all available POI layers in the current map. Users can toggle individual layers to choose which layer is visible on screen.

Clicking on a layer in the menu toggles it.
