\doxysubsection{Detailed Description}
The \doxylink{class_deck_main_window}{Deck\+Main\+Window} class provides a UI for managing decks. 

\doxylink{class_deck_main_window}{Deck\+Main\+Window} extends QWidget to provide a user interface where users can add, edit, and delete decks. It interacts with the \doxylink{class_main_window}{Main\+Window} to reflect these changes across the application.
