\doxysubsection{Detailed Description}
Specialized class for text-\/based questions. 

\doxylink{class_text_question}{Text\+Question} extends the \doxylink{class_question}{Question} class to provide implementations specific to text-\/based questions and answers. It includes methods for displaying the question on a QLabel, updating the UI form with the question and answer text, and providing a unique label for the question.
