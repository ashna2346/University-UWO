This file includes the implementation of \doxylink{class_deck_main_window}{Deck\+Main\+Window}, which handles the main window interface for managing decks in a quiz or flashcard application. It includes functionalities to add, edit, rename, and delete decks, as well as saving and loading deck configurations to and from json files.
