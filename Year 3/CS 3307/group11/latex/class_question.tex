\doxysubsection{Detailed Description}
Abstract base class for quiz questions. 

\doxylink{class_question}{Question} is an abstract class that defines the basic structure and functionality of a quiz question, including getters and setters for the question text, answer, deck name, and difficulty rating. It requires implementation of specific methods for displaying the question and updating forms in the UI.
