\doxysubsection{Detailed Description}
A dialog window for displaying session summaries. 

\doxylink{class_summary_dialog}{Summary\+Dialog} is a specialized QDialog that presents a summary or conclusion text to the user at the end of a session. It contains a QLabel for the summary text and provides a method to set this text.
